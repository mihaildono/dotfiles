# Installation
Clone repo:
``` sh
$ git clone git@github.com:mihaildono/dotfiles.git ~/.emacs
```
For Linux use auto-installer:
```
$ sh ~/.emacs.d/config/install.sh
```
For MacOS check commented installer at:

```
sh ~/.emacs.d/config/install.sh
```
